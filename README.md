# 📊 Разработка A/B-тестирования и анализ результатов

**Проект**  
Проведение полного цикла A/B-теста: от расчета параметров и проверки корректности до статистического анализа результатов и формулировки выводов для бизнеса.

**Роль**  
Продуктовый аналитик

**Стек**  
`Python` `Pandas` `Matplotlib` `SciPy` `Statsmodels`

**Статус**  
✅ Завершен

---

## 📌 Описание проекта

Команда разработки развлекательного приложения (аналог TikTok/Reels) предложила внедрить новый алгоритм рекомендаций, который должен повысить вовлеченность пользователей. Моя задача – спланировать, провести и проанализировать A/B-тест, чтобы принять обоснованное решение о внедрении.

### Целевая метрика
**good_session** – доля сессий, в которых пользователь просмотрел 4 и более страниц. Эта метрика отражает удовлетворенность контентом и качество рекомендаций.

### Гипотезы
- **H₀:** Доля `good_session` в группах A и B **одинакова**.
- **H₁:** Доля `good_session` в группе B (новый алгоритм) **выше**, чем в группе A.

### Параметры теста

| Параметр | Значение |
|----------|----------|
| Уровень значимости (α) | 0.05 |
| Статистическая мощность | 0.8 |
| MDE | 3% |
| Размер выборки | 41 040 пользователей на группу |
| Длительность | 9 дней |

---

## 🔍 Ход работы

### 1. Планирование эксперимента
- Рассчитан необходимый размер выборки с учетом MDE и базового уровня метрики.
- Определена длительность теста на основе среднего дневного трафика.

### 2. Проверка корректности теста
На данных первых дней теста выполнены следующие проверки:
- **Равенство групп** – разница в размерах групп составила менее 1%.
- **Отсутствие пересечений** – ни один пользователь не попал в обе группы.
- **Равномерность по устройствам и регионам** – распределение пользователей по категориям статистически не различается.

### 3. Анализ результатов
- Доля успешных сессий в группе B оказалась **на 1.1% выше**, чем в группе A.
- Проведен **z-тест для пропорций** для проверки статистической значимости.
- **p-value составило 0.00016**, что значительно ниже уровня значимости α = 0.05 → **H₀ отвергается**.
